How to use in Sonarr

  1. Go to Settings → Import Lists
  2. Click Add List → Custom List
  3. Paste one of the URLs from the boxes
  4. Set Monitor to your preference
  5. Click Save — Sonarr imports on its next sync

Lists update weekly. Toggle New only to exclude sequels and spin-offs.
